# golgehatti
CyberAkademi.Org Gölge Hattı Proxy Scraper v1.0

Bilgisayarınızda python kurulu değilse yükleyin. https://www.python.org/downloads/<br>
Aracı bir klasör içine yerleştirip açın.<br>
Otomatik tarama başlayacak ve bittiğinde Gölge Hattı'nda anlık olarak yer alan tüm proxyler klasör içerisine golgehattiproxy.txt olarak kaydedilecektir
